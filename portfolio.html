War Game

During a hackathon in Georgia Tech, my partner and I created an iOS game app based on the classic War card game. I learned Swift and how to put an app together in iOS, both of which I did not know prior. For this app, we each created our own War game so that we could both gain a better overall understanding of the app development process. Of course, it was also helpful since we could help each other if one got stuck at a certain part. Collaborating with another student helped me develop teamwork and communication skills.

Pet Store Web Apps

For this project, I developed a responsive web application that can record, track, and maintain relevant company data. I created a Pet Store database in MySQL to use as for testing and to link to the web application. Java servlets and JSP were used to create the add, delete, and edit functions while jQuery was used for form validation. This project was helpful because it not only improved my SQL skills, but also taught me about MVC framework, XSS, and CRUD functionality.

Currency Converter

The Currency Converter app is an Android app created to convert US dollars to either euros, pesos, or Canadian dollars. The purpose of this app is to learn how to use radio buttons. I learned how to connect them through radio groups and how to code the radio buttons so it converts into the correct currency when the button is selected. This app was also also introduced me to data validation and toast notifications on Android. I also learned how to correctly align each widget so that they stay put for every screen.

Tip Calculator

This Android app was created with the intention to help guest split the bill with an included tip. It can calculate up to ten guest and 25% tip. This app introduced me to Android Studios and helped me gain a better understanding of the Android platform. The main purpose of this project is to get more familiar with Android development by learning about spinner icons and by getting familiar with launcher icons and themes.
